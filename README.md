# MyReader
A concise e-book reader written in rust
# Feature
- Format support:
    - [ ] epub
    - [ ] txt
- Platform support: 
    - [x] Web
    - [x] MacOS
    - [ ] Windows
    - [ ] Linux
- Theme support: 
    - [ ] Light mode
    - [ ] Night mode
    - [ ] Custom mode

# Development
Run dev 
```bash
yarn
yarn tauri dev
```
